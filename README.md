# Hacking The Fender

## Overview

This project demonstrates Python file handling by simulating a simple password security workflow.

The program reads a CSV file containing compromised user information, creates a text file of affected usernames, generates a JSON notification for the boss, and replaces the password file with an ASCII art warning message.

The project focuses on practicing Python's built-in file handling capabilities using context managers (`with open()`), along with the `csv` and `json` standard libraries.

## Features

- Read data from a CSV file using `csv.DictReader`
- Write compromised usernames to a text file
- Create a JSON notification using the `json` module
- Generate a replacement password file containing an ASCII art warning
- Practice safe file handling with `with open()`
